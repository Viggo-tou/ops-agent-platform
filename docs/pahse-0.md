# Phase 0 Scope

## Selected scenarios
1. Employee request -> Jira ticket draft
2. Employee asks process question -> knowledge-based answer
3. Employee action request -> approval checkpoint flow

## Why these scenarios
- close to enterprise agent job descriptions
- easy to demo clearly
- naturally involve tasking, knowledge lookup, permissions, and approvals

## System integration target
- Jira only

## Permission categories
- read-only tools
- write tools
- approval-required tools

## Out of scope
- real external integrations
- multiple enterprise systems
- multi-agent orchestration
- advanced admin analytics
